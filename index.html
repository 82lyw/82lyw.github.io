sass

“CSS 预处理器”，让 CSS 像其它程序语言一样可以做一些预定的处理，为 CSS 增加一些编程的特性。

！default 设置默认值，而覆盖则是再其之前再次声明该变量

使用：
例如把 padding 的值从 2px 改成 5px

编译成 css 后

这种情况使用默认值，可以让我们在不对.scss 源文件进行修改，而对我们要修改的地方进行了修改，同时不会产生重复的代码

@mixin 可以实现大段样式的重用

sass 的逻辑：如@if、@while、@each、@for

移动端的适配

###

eg.

设置理想视口：把默认的layout viewport的宽度设为移动设备的屏幕宽度，得到理想视口(ideal viewport)
width=device-width 这句代码可以把布局视口设置成为浏览器（屏幕）的宽度
user-scalable=no 禁止用户缩放
initial-scale=1 的意思是初始缩放的比例是 1,起到了和width=device-width同样的效果
maximum-scale=1.0, minimum-scale=1.0最大最小的缩放级别

媒体查询@media

min-width、max-width对应的属性值就是响应式设计中的断点值
来解决自适问题

设配 rem

适配原理：将px替换成rem,动态修改html的font-size
1 rem = 浏览器默认的字体大小

Flexible 布局

相对单位 rem 基准值公式：
rem=document.documentElement.clientWidth*dpr/10
将视觉稿分成 100 份，每份被称为一个单位 a
1rem被认为10a
Flexible 会将实际视口宽度除以 10，并将这个值设置为根元素 html 的font-size值。
由此可以推出：对于采用标准iPhone 6的750×1334视觉稿来说
1a = 7.5px 1rem = 75px
基于这个计算结果，我们就能将 1:1 标注稿的 px 数值除以 75 转换成 rem 数值，再套用到 CSS 中。

了解jQuery
